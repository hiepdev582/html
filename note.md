### III. HTML Entities - Character References

| Character | Description           | Code     |
| --------- | --------------------- | -------- |
| <         | Less than             | &lt;     |
| ≤         | Less than or equal    | &le;     |
| >         | Greater than          | &gt;     |
| ≥         | Greater than or equal | &ge;     |
| ≠         | Not equal             | &ne;     |
| ≡         | Identical to          | &equiv;  |
| ≈         | Approximately equal   | &asymp;  |
| ⊂         | Subset of             | &sub;    |
| ⊃         | Superset of           | &sup;    |
| ←         | Left arrow            | &larr;   |
| →         | Right arrow           | &rarr;   |
| ↑         | Up arrow              | &uarr;   |
| ↓         | Down arrow            | &darr;   |
| "         | Double quote          | &quot;   |
| '         | Single quote          | &apos;   |
| °         | Degree                | &deg;    |
| $         | Dollar                | &dollar; |
| ©         | Copyright             | &copy;   |
| ®         | Registered            | &reg;    |
| ™         | Trademark             | &trade;  |
| &         | Ampersand             | &amp;    |
| ±         | Plus-minus            | &plusmn; |
| ×         | Multiplication        | &times;  |
| ÷         | Division              | &divide; |
|           | Space                 | &nbsp;   |
